## エグゼクティブサマリー

本レポートは、KECゼミナールの事業環境・競合優位性・課題を分析し、改善戦略を提案するものである。リサーチ、戦略策定、客観レビューの3段階プロセスを経て作成した。

### 主要な発見

- **最大の強み**：国公立中学受験で関西No.1の合格実績（2025年度394名）。明確なニッチポジショニングで大手塾との棲み分けが成立
- **主要な課題**：料金体系の不透明さ、授業環境の管理、講師の多忙
- **最大のリスク**：少子化の進行と、大手塾の国公立受験コース参入の可能性
- **レビュー総合評価**：3.0/5.0 — 方向性は妥当だが、定量データ・コスト試算の追加が不可欠

### 最優先で取り組むべき3施策（レビュー反映後）

1. **定量データの収集・分析基盤の整備** — 生徒数推移・退塾率・入塾経路等のデータなくして施策の効果検証は不可能
2. **講師の負荷軽減と採用強化** — 多くの課題の根本原因。既存SaaS（Comiru等）活用で事務負荷を削減
3. **合格実績マーケティングの強化** — 「国公立中学合格 関西No.1」は低コストで即効性の高い訴求ポイント

## 3. 競合優位性

KECゼミナールは「国公立中学受験」というニッチ市場でリーダーポジションを確立。

**持続的な競争優位の源泉：**
- 長年蓄積されたKECCADASデータ（参入障壁）
- 表現力・思考力教育の専門ノウハウ
- 奈良県北部での保護者口コミネットワーク

**レビュアーからの注意点：**
- 「模倣困難」の根拠が不十分 — エデュテイメント等は概念としては他塾も導入可能
- 大手塾の国公立コース参入の実態調査が必要

> 詳細は [02_strategy.md](./02_strategy.md) セクション2を参照

## 4. 改善戦略（レビュー反映版）

レビュアーの指摘を反映し、施策をフェーズ1は**3つに厳選**した。

### フェーズ1：基盤整備（2026年4月～9月）

| 優先度 | 施策 | 具体的アクション | コスト感 |
|--------|------|-----------------|---------|
| 1 | 定量データ収集基盤の整備 | 生徒数推移・退塾率・入塾経路・保護者満足度調査の実施。全施策のPDCA基盤を構築 | 低（内部工数中心） |
| 2 | 講師負荷軽減 | 塾向けSaaS（Comiru等）導入で事務・連絡業務を効率化。宿題管理・保護者連絡を一元化 | 中（月額数万円/教室） |
| 3 | 合格実績マーケティング強化 | 「国公立中学合格 関西No.1」のWeb・SNS発信強化。合格体験記・保護者インタビューの制作 | 低～中 |

### フェーズ2：品質向上・ブランド強化（2026年10月～2027年9月）

| 施策 | 内容 |
|------|------|
| 授業環境管理の強化 | ルール明文化、パイロット教室での試行後に横展開 |
| 料金の「納得感」醸成 | 単なる透明化ではなく、投資対効果のストーリー構築 |
| 保護者セグメント別アプローチ | 価格重視層・実績重視層・理念共感層ごとのコミュニケーション設計 |
| 低学年プログラム拡充 | 早期囲い込みによるLTV向上 |

### フェーズ3：事業拡大（2027年10月～2029年3月）

| 施策 | 内容 |
|------|------|
| 新規エリア進出 | 投資回収計画を策定した上で、1教室ずつ慎重に拡大 |
| 少子化対応の事業多角化 | 表現力講座の外販、公立高校受験コース強化 |
| DX推進 | データ蓄積状況を踏まえたAI学習診断の検討 |

> 詳細は [02_strategy.md](./02_strategy.md) セクション4-6を参照

```
2026/4          2026/10         2027/4          2027/10         2028/4          2029/3
  |--- フェーズ1 ---|--- フェーズ2 ---------|--- フェーズ3 --------------------|

  [データ収集基盤]
  [SaaS導入・講師負荷軽減]
  [合格実績マーケ強化]
                    [授業環境改善(パイロット→横展開)]
                    [料金納得感醸成]
                    [セグメント別アプローチ]
                                    [新規エリア進出検討→実行]
                                    [事業多角化]
                                    [DX推進]
```

**重要な変更点（レビュー反映）：**
- フェーズ1の施策を7つ→3つに絞り込み
- パイロット教室方式の採用（全教室一斉展開を回避）
- アプリ自社開発→既存SaaS活用に変更（コスト大幅削減）
- 講師採用・定着策を中期→短期に前倒し

## 6. レビュー指摘事項と対応

| レビュー指摘 | 深刻度 | 対応方針 |
|-------------|--------|---------|
| 定量データの欠如 | 最重要 | フェーズ1の最優先施策として組み込み済み |
| コスト試算の不在 | 最重要 | 各施策にコスト感を追記。詳細試算は内部データ入手後に実施 |
| 施策過多（組織キャパ超過リスク） | 高 | フェーズ1を3施策に厳選 |
| 「料金透明化→口コミ改善」の因果の飛躍 | 高 | 「透明化」→「納得感の醸成」にアプローチ変更 |
| 講師負荷への配慮不足 | 高 | 講師負荷軽減を最優先施策に格上げ |
| 教育政策変動リスクの未考慮 | 中 | モニタリング体制の構築を推奨事項に追加 |
| アプリ自社開発の費用対効果 | 中 | 既存SaaS活用に変更 |

## 7. 今後のアクション

### 即座に着手すべきこと

1. KECゼミナールの内部データ（財務・生徒数・退塾率・講師数）の入手・分析
2. 塾向けSaaS（Comiru等）の比較検討・トライアル
3. 「国公立中学合格 関西No.1」訴求のWebコンテンツ制作着手

### 追加調査が必要な項目

- 奈良県・大阪府の児童数推移と国公立中学の定員・倍率の推移
- 大手塾（馬渕教室等）の国公立受験コースの動向
- 保護者の入塾意思決定プロセスの実態調査

### 意思決定に必要な情報

- KECゼミナールの年間売上・利益率 → 投資可能額の算定
- 講師の現在の業務負荷の定量評価 → 施策の実行可能性判断
- 各教室の定員充足率 → 拡大戦略の妥当性検証
