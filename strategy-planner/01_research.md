# KECゼミナール 調査レポート

調査日：2026年4月1日

## 1. 事業概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | KECゼミナール・KEC志学館ゼミナール |
| 運営会社 | 株式会社ケーイーシー（KECグループ） |
| 本部所在地 | 奈良県生駒市 |
| 創立 | 1977年（法人化：1982年） |
| 対象学年 | 小学1年生～中学3年生 |
| 授業形式 | 集団指導 |
| 教室数 | 約18教室 |
| 展開地域 | 奈良県・大阪府・京都府 |

### 沿革・経緯

- 1977年に創立、1982年に法人化
- 奈良県北部を中心に教室を展開
- 2016年に奈良県南部に拠点を持つ株式会社アンドリュー（志学館）と経営統合し、「KECゼミナール・KEC志学館ゼミナール」の体制となる

### 主な教室

生駒教室、学園前教室、西大寺教室、登美ヶ丘教室、高の原教室、奈良教室、新大宮教室、木津南教室、京田辺教室、郡山教室、桜井教室、五位堂教室、王寺教室、天王寺教室、上本町教室、神宮教室、八木教室、高田教室

## 2. 教育サービスの特徴

### 企業理念・教育コンセプト

- 企業理念：「人間大事の教育」
- 教育コンセプト：「子どもたちの10年・20年先にも続く自信を育てる」
- 詰め込み指導ではなく、自主性や人間力を磨くことで成績向上・受験合格を目指す

### 指導方法

- **エデュテイメント授業**：楽しみながら学習を進める授業スタイル。「楽しい授業が生徒の能力を引き出す」という理念に基づく
- **対話形式の授業**：生徒を積極的に参加させ、一人ひとりと密にコミュニケーションを図る。自分の意見を自分の言葉で表現する力を育成
- **KECCADAS**：長年にわたり奈良の中学校テストを分析したデータをもとにした、学校別の定期テスト対策プログラム

### カリキュラム

- **先取り中心のカリキュラム**：常に二手先をいくことで志望校合格へ導く
- **G-PDCAサイクル**：目標設定（Goal）→ 学習プラン（Plan）→ 実践（Do）→ テスト検証（Check）→ 改善策（Action）の5ステップで学習を進める
- **表現算数・表現国語**：国公立附属中入試に必要な表現力を磨く独自カリキュラム
- **KECオリジナル模試**：合格力を測定する独自の模擬試験

### クラス編成

- 集団指導を基本としつつ、学力レベルや志望校に応じたクラス編成
- 中学受験専門コースあり（天王寺教室など）

## 3. 強み

### 国公立中学受験に圧倒的な強さ

- 2025年度は関西No.1の394名が国公立中学に合格
- 国公立中学8校で合格者数1位を達成
- 奈良女子大学附属中等教育学校、大阪教育大学附属天王寺中学校などで定評あり

### アットホームな指導環境

- 大手進学塾とは異なり、アットホームな雰囲気
- 講師との距離が近く、自習室でもすぐに質問できる環境
- 生徒一人ひとりへのきめ細かい対応

### 奈良県に特化した地域密着型

- 奈良県の中学校テストを長年分析した独自データベース（KECCADAS）
- 地域の学校事情に精通した指導
- 奈良県北部を中心に高いブランド認知度

### 独自の教育メソッド

- エデュテイメント授業による学習への動機づけ
- G-PDCAサイクルによる自律的な学習習慣の確立
- 表現力重視のカリキュラム（表現算数・表現国語）

## 4. 弱み・課題

### 料金面の不満

- 中学生の授業料は安いとは言えない水準
- 月謝以外の費用（季節講習、教材費、施設維持費など）が高いという口コミが多い
- 春季・冬季講習は通常授業とほぼ同じ内容にも関わらず追加費用が発生するとの不満あり
- 当初の予想よりも総額が割高になりがち

#### 料金目安

| 項目 | 金額（税込） |
|------|-------------|
| 入学金 | 16,500円 |
| 小学生月額 | 約11,000円～ |
| 中学生月額 | 約13,750円～ |

※受験対策コースや特別指導は別途料金

### 授業環境の課題

- 授業中に騒がしい生徒がいても講師がなかなか注意しないという指摘
- 集中しにくい環境になることがある

### 講師の忙しさ

- 宿題の範囲が分かりにくい場合がある
- 講師が忙しそうで子どもが質問しにくいタイミングがある

### 展開地域の限定

- 奈良県中心の展開であり、大阪府・京都府への進出は限定的
- 全国的な知名度は大手塾と比較して低い

## 5. 評判・口コミ

### 肯定的な評価

- **講師の質**：「先生たちの教え方が面白い」「わかりやすい」と高評価
- **質問しやすい環境**：「自習室で分からないことがあればすぐに聞きに行ける」
- **合格実績**：国公立中学受験において安定した実績
- **子どもの成長**：表現力やプレゼン能力が身についたという声
- **オリコン顧客満足度調査**：中学受験（集団塾）近畿ランキングでも評価を得ている

### 否定的な評価

- **料金面**：「月謝以外の出費が多い」「季節講習の追加費用が不満」
- **授業環境**：「騒がしい生徒への対応が甘い」
- **コミュニケーション**：「宿題の範囲が分かりにくい」「講師に質問しにくいときがある」

### 総合評価

全体としては講師の質や合格実績に対する満足度が高い一方、料金体系の透明性や授業中の環境管理に改善の余地があるという傾向が見られる。

## 6. 競合環境

### 奈良県・関西圏の主要競合塾

| 塾名 | 特徴 | KECとの比較 |
|------|------|------------|
| **馬渕教室** | 関西最大級の進学塾。私立中学受験に強い。費用は中程度（小4で年間約56万円～） | KECは国公立中受験に特化、馬渕は私立中受験に強い。馬渕のほうが教室数・知名度で上回る |
| **浜学園** | 関西屈指の難関校対策塾。カリキュラムの進度が早い。テスト回数が多い。費用は高め | KECはアットホーム、浜学園はハイレベル・競争型。対象層が異なる |
| **能開センター** | 講師評価が高い（オリコン75.4点）。幅広い学力層に対応 | KECと同様にきめ細かい指導が特徴。競合度が高い |
| **希学園** | 最難関校特化。費用は関西の塾で最も高い | 対象層が異なる（希は最難関私立、KECは国公立中心） |
| **日能研** | 全国展開の大手。中堅校から難関校まで幅広くカバー | KECは地域密着、日能研は全国ブランド |

### KECゼミナールの市場ポジション

- **ニッチ戦略**：国公立中学受験という明確なポジショニングで差別化
- **地域密着**：奈良県内では強固なブランドを持つが、関西全域では大手に知名度で劣る
- **価格帯**：中程度だが、総費用では割高感を指摘する声もある
- **競合との棲み分け**：私立難関校を目指す層は浜学園・希学園・馬渕教室へ、国公立中学を目指す層はKECゼミナールへという棲み分けが成立している

---

## 7. まとめ

KECゼミナールは、奈良県を拠点に国公立中学受験で関西No.1の実績を持つ地域密着型の学習塾である。「人間大事の教育」を理念とし、詰め込みではなくエデュテイメント授業やG-PDCAサイクルなど独自の教育メソッドで、子どもの自主性と表現力を育てることに注力している。

強みは国公立中受験の圧倒的な合格実績、アットホームな環境、奈良県に特化した定期テスト対策（KECCADAS）である。一方、課題としては料金の不透明さ（季節講習等の追加費用）、授業中の環境管理、展開地域の限定などが挙げられる。

競合環境においては、私立難関校向けの大手塾（浜学園・馬渕教室・希学園）とは国公立中学受験という軸で明確に棲み分けており、このニッチな市場でのリーダーポジションを確立している。
